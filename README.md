Dinner
======

Invitez vos amis à manger MAIS
- pas tous en même temps 
- pas deux fois la même chose
 


AMIS
======
Une personne se caracterise par :
- un identifiant
- une ou des photos
- son nom
- son prenom
- son commentaire


RECETTE
======
Une recette se caracterise par :
- un identifiant
- une ou des photos
- un nom
- une liste d'ingredient


REPAS
======
Une repas se caracterise par :
- un identifiant
- une liste de recettes
- une liste de personne
- une liste de note/personne/recette
